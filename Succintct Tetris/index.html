How to Play

Tetris is a game where you try to create complete lines by directing falling blocks.

  • ← → Keys: Move blocks left or right
  • ↑ Key: Rotate block counterclockwise
  • ↓ Key: Rotate block clockwise
  • Space Key: Drop block quickly

The game lasts for 2 minutes. Try to get the highest score in this time!
